Simple Linear Regression¶

An alternative way to do linear regression is to use statsmodels.formula.api. The syntax here is similar to R. The advantage is that it provides a nice summary of statistical properties of the model.

results = smf.ols('Sales ~ TV', data=adv).fit()

The summary contains all the value of parameters, t-statistics, F-statistics. Also, it provides 95% confidence interval of each parameter. Let's verify whether the 95% interval is calculated correctly. From the textbook, we know: $$\beta_1 \in \left[\hat{\beta}_1 - t_{1-\alpha/2,n-2} \textrm{SE}(\hat{\beta}_1), \hat{\beta}_1 + t_{1-\alpha/2,n-2} \textrm{SE}(\hat{\beta}_1)\right]$$ and $$\beta_0 \in \left[\hat{\beta}_0 - t_{1-\alpha/2,n-2} \textrm{SE}(\hat{\beta}_0), \hat{\beta}_1 + t_{1-\alpha/2,n-2} \textrm{SE}(\hat{\beta}_0)\right]$$

In [10]:
import scipy.stats as ss

Note that ss.t.isf function is the inverse of 1-F(x). Therefore, $\text{isf}(\alpha/2) = F^{-1}(1-\alpha/2)$
